---
description: How to start all FGSN services simultaneously
---

To start the entire FGSN ecosystem (8 services), run the following command from the root directory:

// turbo
```powershell
npm.cmd run dev
```

### Port Mapping Reference:
- **3000**: Auth Gateway
- **3001**: Admin Governance
- **3002**: Scoring Engine
- **3003**: Broadcast Overlay
- **3004**: Public Portal
- **3005**: Socket Relay
- **3006**: Manager Tactical Hub
- **3007**: Scorer Personnel Hub
